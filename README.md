# vibe_code_starter

Reusable AI-assisted software engineering starter framework for Claude Code + VSCode.

## Workflow

1. Create new repo from this template
2. Open in VSCode
3. Start Claude Code
4. Tell Claude:

"Read CLAUDE.md and run the project setup workflow before writing code."

5. Complete project intake
6. Approve architecture/build plan
7. Begin implementation

## Core Files

- CLAUDE.md → repo operating instructions
- PROJECT_BRIEF.md → project goals and constraints
- BUILD_PLAN.md → implementation roadmap
- DECISIONS.md → architecture decisions and rationale
- REVIEW_CHECKLIST.md → production/review standards
